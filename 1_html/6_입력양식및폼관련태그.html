폼 관련 태그

form 태그 내의 submit 버튼 클릭 시 해당 form 내에 작성된 사용자가 입력한 값을 서버로 넘기면서 요청(request)하는 역할을 수행한다

[표현법]
form action="서버주소" method="GET/POST"
* action : form 태그 내부의 입력한 값들을 전송할 서버의 주소
* method : 서버로 데이터 요청 시 전송방식을 지정하는 속성
  1. GET : 사용자가 전달하는 값들을 URL에 노출하여 전송하는 방식
  2. POST : 사용자가 전달한 값을 URL BODY 영역에 담아 전송하는 방식

라디오버튼과 체크박스

라디오 버튼

- 제시한 값 중 1개만 선택해야하는 경우 사용
- 서버로 요청 시 선택된(checked) 라디오버튼의 값이 전송된다
- name 속성을 통해 radio 버튼을 그룹으로 만듬.

체크박스

- 제시한 값 중 여려개를 선택하는 경우 사용
- 선택된 체크박스의 값들을 서버로 전송
- name 속성을 통일시켜 하나의 그룹으로 만듬.

select 태그와 option 태그

- 사용자에게 직접 키보드로 값을 입력받는 것이 아닌, 제시한 여러 옵션들 중 하나를 선택할 수 있도록 하는 태그
- 서버로 요청 시 현재 선택된(selected) 값이 서버로 전달됨
- 각 option 태그에 value 속성을 작성한 경우 값이 서버로 전달되고, 명시하지 않은 경우 content 영역의 값이 서버로 전송됨
