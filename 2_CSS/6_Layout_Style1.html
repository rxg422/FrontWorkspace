화면 배치 방법 : display

        요소가 화면에 출력되는 방법을 지정

        - block : 화면을 수직 분할(행)
        - inline : 화면을 수평 분할(열)
        - inline-block : inline의 수평 분할 + block의 크기조절 기능
        - none : 화면에 요소를 출력하지 않음
        - flex : 요소의 정렬되는 방향이나 간격을 유연하게 처리
