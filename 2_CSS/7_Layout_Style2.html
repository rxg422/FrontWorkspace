요소의 영역관련 속성

        HTML의 요소는 총 4가지 영역으로 구성

        1. content 영역
            - 요소의 내용이 작성되는 영역
            - 시작 태그와 종료 태그 사이에 작성

        2. padding 영역
            - content 영역과 border 영역 사이에 존재

        3. border 영역
            - 요소의 테두리 영역
            - content 영역을 감싸고 있다.

        4. margin 영역
            - 현재 요소와 다른 요소와의 간격
