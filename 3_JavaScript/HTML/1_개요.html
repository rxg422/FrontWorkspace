JavaScript 개요

        - html의 조작과 변경을 담당하는 언어
        - html로 웹문서 구조 제작, css로 디자인 정의, js로 웹페이지상의 이벤트 발생 시 동작 구현
        - 웹 브라우저에서 실행 가능한 인터프리터 방식(컴파일 없이 한줄 씩 실행)의 스크립트 언어

        장점 
            - 코드 작성 후 실행결과를 바로 확인할 수 있어 간단한 개발에 용이
            - 클래스, 접근제한자, 반환형 들 고급 개념없이 바로 사용가능하여 초보자가 다루기 쉽다.
            - 모든 웹브라우저의 표준 스크립트 언어임으로 범용성이 넓음.

        단점
            - 실행 속도가 느림.
            - 큰규모의 어플리케이션 개발에 있어서 타입안전성이 부족

1. JS 사용 방법

  1. inline : 이벤트를 부여하고자 하는 요소 내에 소스코드를 작성하는 방법
  2. internal : html문서 내에 script 태그 영역에 작성하는 방법
  3. external : 소스코드를 별도의 .js로 작성하여 불러오는 방법
