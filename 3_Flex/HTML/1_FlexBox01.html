Flex Box

        요소의 정렬되는 방향, 순서, 요소 간격을 수치적으로 처리하지 않아도 알아서 유연하게 처리해주는 방식

        * Flexbox 핵심 개념

        1. Flexbox 구성요소
            - flex-container : 정렬이 필요한 요소를 감싸는 상위 컨테이너
            - flex-item : 정렬이 적용되는 요소

        2. Flexbox 정렬 축
            - main axis : 중심 축
            - cross axis : 교차 축
